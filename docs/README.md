## üîß Core Concepts

### Schema Compression
Reduces dataset representation by 50-2000x while maintaining semantic meaning.
See: [ARCHITECTURE.md - Schema Compression](../ARCHITECTURE.md)

### History Compression
Compresses analysis history by 5-20x while preserving insights.
See: [ARCHITECTURE.md - History Compression](../ARCHITECTURE.md)

### EDA Agent
AI-powered agent that automates exploratory data analysis.
See: [API_REFERENCE.md - EDAAgent](API_REFERENCE.md)

### ScaleDown Integration
Ultra-compression layer using ScaleDown API.
See: [API_REFERENCE.md - ScaleDownIntegration](API_REFERENCE.md)
